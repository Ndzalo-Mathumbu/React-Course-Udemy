# The Wild Oasis

A modern hotel management dashboard built with React and Supabase.

## Overview

The Wild Oasis is a hotel management application that allows hotel staff to manage cabins, bookings, guests, check-ins, check-outs, and user accounts through a clean and responsive dashboard.

This project was built while following Jonas Schmedtmann's React course. While the project structure and core features were guided by the course, building it helped me gain valuable hands-on experience with modern React development and real-world application architecture.

## Note

This application is not entirely dynamic.

Some parts of the project use static data that was provided as part of the learning process, while other sections are connected to Supabase and work with real database data.

The purpose of the project was to learn modern React development practices and understand how production-style applications are structured.
